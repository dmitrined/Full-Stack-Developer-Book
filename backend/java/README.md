# Java Core Index ☕

Этот раздел посвящен основам языка Java — фундаменту, на котором строится Spring Boot и любой современный корпоративный софт.

## Основные темы (Подробные конспекты)

1.  **[Core Basics](./CoreBasics.md)** 🧠 — Память (Stack/Heap), String Pool, контракты Object.
2.  **[OOP & SOLID](./OOP_SOLID.md)** 🏗️ — Принципы проектирования и основы ООП.
3.  **[Collections Framework](./Collections.md)** 📚 — Списки, Сеты, Мапы и их внутренняя работа.
4.  **[Multithreading](./Multithreading.md)** 🧵 — Многопоточность, JUC и Virtual Threads.
5.  **[Functional Programming](./StreamAPI_Functional.md)** ⚡ — Lambdas, Stream API и Optional.
6.  **[Generics & Reflection](./Generics_Reflection.md)** 🔍 — Параметризация и работа с метаданными.

---

### Почему это важно?
Без понимания того, как работает `HashMap` внутри или как JVM управляет памятью, невозможно писать высокопроизводительные приложения на Spring Boot. Эти темы являются обязательными на интервью уровня Middle/Senior.
