# Шпаргалки и Горячие клавиши ⚡

## IntelliJ IDEA / Android Studio (Mac)
| Действие | Клавиши |
| :--- | :--- |
| **Search Everywhere** | `Shift` `Shift` |
| **Find Action** | `Cmd` + `Shift` + `A` |
| **Project Structure** | `Cmd` + `;` |
| **Generate Code** (G/S/Cons) | `Cmd` + `N` |
| **Reformat Code** | `Cmd` + `Opt` + `L` |
| **Optimize Imports** | `Control` + `Opt` + `O` |
| **Quick Documentation** | `F1` |
| **Basic Completion** | `Control` + `Space` |

## Visual Studio Code (Mac)
| Действие | Клавиши |

| **Command Palette** | `Cmd` + `Shift` + `P` |
| **Go to File** | `Cmd` + `P` |
| **Toggle Sidebar** | `Cmd` + `B` |
| **Integrated Terminal** | `Control` + `` ` `` |
