# E2E UI Automation 🖱️

Автоматизация действий пользователя в браузере.

## 1. Selenium WebDriver
Классика автоматизации. Работает через драйверы браузеров.
- `driver.get(url)`
- `findElement(By.id("..."))`
- `click()`, `sendKeys()`

## 2. Playwright 🎭
Современный инструмент от Microsoft. Быстрее и надежнее Selenium.
- Встроенное ожидание (Auto-waiting).
- Запись видео и трейсов.
- Поддержка нескольких вкладок и контекстов.

## 3. Page Object Pattern (POM)
Паттерн проектирования, где страницы сайта описываются как отдельные классы с методами-действиями. Это делает тесты читаемыми и легкими в поддержке.
